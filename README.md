ProjectFarmCapiz is an initiative of the Capiz Innovation Center (CIC) to digitize farming practices across the province.
Our goal is to empower farmers and farm owners with affordable, low-maintenance, long-range sensors that feed real-time data into a centralized database. This data powers dashboards and condition alerts, enabling smarter decisions on labor allocation, resource use, and farm management.
